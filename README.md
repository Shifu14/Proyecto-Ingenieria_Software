SGCU - Sistema de Gestión de Comedor Universitario

Este proyecto es una aplicación de escritorio diseñada para la administración y vista del comedor universitario de la UCV, permite el manejo de usuarios mediante login, la vista del menu del comedor, el calculo de costos.

Tecnologías y Librerías Utilizadas

El proyecto ha sido desarrollado utilizando herramientas nativas de Java:

- Lenguaje: Java JDK 17.
- Librerias de interfaz grafica: javax.swing y java.awt. Se utilizaron componentes como JFrame, JPanel, y LayoutManagersl.
- Gestión de Datos: Manejo de archivos planos y formato JSON para la persistencia de datos (Costos y Menú).
